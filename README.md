예약 구매 프로젝트
==================

일반 상품과 예약 상품을 등록하고 이용자들이 등록된 상품을 구매할 수 있는 프로젝트입니다.

## 도커 컴포즈 실행
* 실행 명령어
```
docker-compose up
```
* 빌드 후 실행 명령어
```
docker-compose up --build
```

## Module

| Service | Description | Port  |
|---------|---------|---------|

| Purchase | 상품 구매 및 결제 관리  | 8082  |
| User  | 회원 정보 관리  | 8083  |
